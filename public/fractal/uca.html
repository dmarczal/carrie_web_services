Usuários dos laptops UCA

A versão do plugin Java usada no seu navegador é incompatível com este programa, por favor sigue as instruções abaixo para atualizar o plugin do Java

  1. Abra o terminal:

    Para isso, clique em metasys -> Aba Favoritos -> Terminal

  2. No terminal copie e cole o seguinte comando:

    sudo ln -sf /usr/java/jre1.6.0_10/plugin/i386/ns7/libjavaplugin_oji.so /usr/lib/browser-plugins/libjavaplugin.so

  3. Quando pedir, a senha digite: metasys
